# Quickstart: Exam Analysis Feature

## Overview
이 기능은 이미지를 포함한 시험 문제를 분석하거나 텍스트 질문을 검색하는 UI를 제공합니다.

## Setup
1. 의존성 설치 (이미 완료됨): `npm install`
2. 개발 서버 실행: `npm run dev`

## Usage

### 1. 이미지 분석 (Exam Analysis)
1. 상단 네비게이션 또는 탭에서 **"이미지 분석"** 선택.
2. **이미지 업로드**: '파일 선택' 버튼을 클릭하여 PNG/JPG 이미지 선택 (1MB 이하).
3. **질문 입력**: 텍스트 입력창에 관련 질문 입력.
4. **분석 요청**: '분석하기' 버튼 클릭.
5. **결과 확인**: 잠시 후 답변, 신뢰도, 관련 문맥 카드가 표시됨.

### 2. 일반 검색 (Text Search)
1. **"일반 검색"** 탭 선택.
2. **질문 입력**: 검색하고 싶은 내용 입력.
3. **검색 요청**: '검색' 버튼 클릭.
4. **결과 확인**: 답변 텍스트 확인.

## Troubleshooting
- **401 Unauthorized**: 로그인이 만료되었을 수 있습니다. 로그아웃 후 다시 로그인해주세요.
- **파일 크기 오류**: 이미지가 1MB를 초과하는지 확인하세요.
